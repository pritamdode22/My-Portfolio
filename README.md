# 💼 Personal Portfolio Website

A responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my projects, technical skills, certifications, and contact information.

This portfolio is designed to present my profile as a **Software Developer / Fresher** in a clean and professional way.

---

## 🚀 Features

- Responsive and modern UI design
- Smooth scrolling navigation
- Project showcase section
- Skills and certifications section
- Contact information section
- Simple animations using JavaScript
- Easy to customize and deploy

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Styling:** Custom CSS (Flexbox, Grid)  
- **Tools:** VS Code, GitHub
